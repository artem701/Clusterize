Кластеризация точек в n-мерном Евклидовом пространстве

Приложение состоит из двух программ: вычислительной части (C++), и UI части (C#)

Вычислительная часть:
class Cluster<T> — кластеризация элементов произвольного типа
Требования к T: наличие функций нахождения расстояния, центра тяжести, оператора < и функции вывода в поток

Выбран иерархический алгоритм снизу вверх, слияние производится невзвешенным центроидным методом
Краткое описание алгоритма:
1) Инициализация списка листьев списком элементов типа Т
2) Подсчет расстояний между всеми листьями
3) Найти два ближайших кластера и объединить, объединенный кластер добавить в список, исходные удалить
4) Если в списке осталось более одного кластера, скорректировать структуру,
   хранящую расстояния между центрами тяжести, вернуться к шагу 3
   
Начальный расчет расстояний потребует n(n+1)/2 проходов
Промежуточный пересчет расстояний после слияния при k элементах потребует k-2 итераций, 
таким образом пока не останется только один кластер пройдет (n-1)(n-2)/2

Ожидаемая сложность алгоритма = O(n^2)

UI часть:
Окно разбито на две вкладки, вторая позволяет заносить информацию о точках в файл,
первая инициирует применение вычислительной программы к выбранному файлу, и загружает результат.
Визуальное представление результата: позволяет выбрать от одного до нескольких отображаемых кластеров, выделенных разными цветами.
С помощью выпадающего списка выбирается, какие координаты точек проецируются на ось X и Y.
Начальный масштаб выбирается таким образом, чтобы разброс точек в выбранной проекции полностью попадал в видимую область экрана.
Масштаб может быть отрегулирован щелчками мыши.
